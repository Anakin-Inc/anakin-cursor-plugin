# Changelog

## 1.0.0

- Initial release of the `anakin` plugin for web data extraction via anakin-cli.
- Added `scrape-website`, `scrape-batch`, `search-web`, and `deep-research` skills wrapping anakin-cli commands.
- Added `anakin-setup` and `anakin-cli-usage` rules for CLI authentication and usage guardrails.
- Added `data-extraction-architect` agent for planning extraction strategies.
